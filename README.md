# Haskell book

This is repo for the [Programiranje u Haskelu](https://haskel.ubavic.rs). The book is written in custom format `.atex` and compiled to `.html` via custom software written in Haskell.

*Programing in Haskell* is written on Serbian, and it is intended as an introductory text to Haskell programming.

This monorepo contains following packages:

+ `book`: text source for the book
+ `vscode-atex`: VS Code extension for `.atex` format
